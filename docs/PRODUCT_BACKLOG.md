# Product Backlog: Crypto Investment Manager

This document outlines the features and requirements for the Crypto Investment Manager, ordered by priority.

| ID | Title | Description | Priority | Effort (Points) | Status |
|----|-------|-------------|----------|-----------------|--------|
| P1 | Authentication | User registration, login, and secure password management. | High | 5 | Completed |
| P2 | Dashboard | Real-time overview of current portfolio and market trends. | High | 8 | In Progress |
| P3 | Risk Analysis | Algorithmic risk evaluation for individual crypto assets. | High | 13 | In Progress |
| P4 | Portfolio Tracker | Manual or automated input of holdings and cost basis. | Medium | 8 | In Progress |
| P5 | Reports (PDF) | Automated generation of performance and risk reports. | Medium | 5 | In Progress |
| P6 | Security Alerts | Real-time notifications for login updates or price triggers. | Medium | 5 | Planned |
| P7 | Settings Management | User preferences for currency and regional settings. | Low | 3 | Planned |
| P8 | Multi-Currency | Support for multiple base default currencies (USD, EUR, INR). | Low | 8 | Planned |
| P9 | 2FA (Advanced) | Implementation of multi-factor authentication for higher security. | High | 13 | Planned |
| P10| API Integration | Direct integration with exchanges (Binance, Coinbase). | High | 21 | Backlog |
