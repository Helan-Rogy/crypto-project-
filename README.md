# Crypto Investment Manager: Agile Documentation

This workspace includes a suite of Agile documents to guide the development and management of the Crypto Investment Manager dashboard.

## Project Documents Index

| Document | Purpose |
|----------|---------|
| [Milestone Roadmap](docs/MILESTONES.md) | **Development progress and implementation status.** |
| [Product Backlog](docs/PRODUCT_BACKLOG.md) | A prioritized list of features and requirements. |

## Quick Start
1. Ensure you have the required Python modules installed.
2. Read the [User Stories](docs/USER_STORIES.md) to understand current feature requirements.
3. Check the [Sprint Plan](docs/SPRINT_PLAN.md) for the latest tasks and progress.
4. All completed work must meet the [Definition of Done](docs/DEFINITION_OF_DONE.md).

## Project Mission
To provide investors with a clean, high-end dashboard for real-time crypto portfolio tracking, risk evaluation, and reporting—built with professional-grade design and robust security.
